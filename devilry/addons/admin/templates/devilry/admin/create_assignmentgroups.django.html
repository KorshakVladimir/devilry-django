Format explanation

The format used to create the assignment groups is:

  • One assignment group on each line.
  • Each username is separated by a comma.
  • Group name is identified by two colons at the end of the name, and must be placed at the beginning of the line.
  • A group name or at least one username is required for each group.
  • An optional candidate id for a candidate is denoted by a colon and the candidate id after the username.

Example: 

user1
user2:5
user3:9, user4, user5:10
Group1::
Group2::user5
Group3::user6, user7
Group4::user8:20, user9:21

Gives the following assignment groups:

  1. Candidates: user1
  2. Candidates: user2 (with candidate id 5)
  3. Candidates: user3 (with candidate id 9), user4 and user3 (with candidate id 10)
  4. Name: Group1
  5. Name: Group2, Candidates: user5
  6. Name: Group3, Candidates: user6 and user7
  7. Name: Group3, Candidates: user8 (with candidate id 20) and user9 (with candidate id 21)
