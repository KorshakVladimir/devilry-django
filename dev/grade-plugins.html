Grade plugins¶

WARNING: OUTDATED docs. This will be updated for the new grade editors.

A grade plugin is a way to plug-in custom grade-handling. We will explain how to create a grade plugin by explaining the most simple grade plugin used in Devilry, devilry.apps.grade_default. grade_default simply allows for any string between 1 and 15 characters to be used as a grade.

Information about a grade plugin.

All parameters are stored as object attributes.

Parameters:
  • view – The view used when creating/editing feedback with this grade-plugin.
  • model_cls – A class for storing the grade of a single Feedback. It us used in a one-to-one relationship with devilry.core.models.Feedback (The Feedback-class takes care of the relationship).
