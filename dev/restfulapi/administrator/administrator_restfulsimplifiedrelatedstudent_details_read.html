GET /administrator/restfulsimplifiedrelatedstudent/id¶

Retrieve a related student.

The underlying data model where the item is retrieved from is defined in devilry.apps.core.models.relateduser.RelatedStudent. The Simplified API that the server forwards this request to is devilry.apps.administrator.simplified.simplifiedrelatedstudent.SimplifiedRelatedStudent.read().

Request¶

Example¶

GET /administrator/restfulsimplifiedrelatedstudent/id

Required request parameters encoded in the URL¶

id¶

The unique identifier of a related student. You will typically get this id in response from a search.

Response¶

On success¶

Example¶

200 OK

{
    id: 15,
    period: 15,
    user: 15,
    tags: "myvalue",
    user__username: "My example value",
    user__devilryuserprofile__full_name: ["this", "is", "an", "example"],
    user__email: "My example value",
    candidate_id: "My example value"
}
