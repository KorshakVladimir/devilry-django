.. _administrator_restfulsimplifiedassignmentgroup_details_delete:

=============================================================================
DELETE /administrator/restfulsimplifiedassignmentgroup/id
=============================================================================

Delete a assignment group.

The underlying data model where the item is deleted from is defined in :class:`devilry.apps.core.models.AssignmentGroup`.


********
Request
********

Example
################

.. code-block:: javascript

    DELETE /administrator/restfulsimplifiedassignmentgroup/id



Required request parameters
###########################

id
--------------

The unique identifier of a assignment group. You will typically get
this id in response from a search.


**************
Response
**************


On success
##########

As long as the assignment group is deleted without an error, the
response is *HTTP 200* with a JSON object containing the id in the response body.


Example
-------

.. code-block:: javascript

    200 OK

    {
        id: 10
    }


On error
########

On errors, we respond with one of the :ref:`restful_api_error_statuscodes`
