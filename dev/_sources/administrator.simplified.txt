:mod:`devilry.apps.administrator.simplified` --- Simplifed API for administrators
=================================================================================

The administrator simplified API. This API is a simplified wrapper of
:ref:`devilry.apps.core.models` for administrators. Every action requires
the given user to be administrator on the object being manipulated.
